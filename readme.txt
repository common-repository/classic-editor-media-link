== Description ==

= Classic Editor Media Link = 

Plugin adds Media Library Button to the default TinyMCE (Classic Editor) Insert Link dialog.

It allows you to choose media file from your library directly in Insert Link dialog.

This plugin works with "Classic Editor", ( for example, used in ACF ), and will not work with Gutenberg Editor.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/classic-editor-media-link` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
